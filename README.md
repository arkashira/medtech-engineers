# Engineer Vetting and Onboarding

This project provides a comprehensive vetting process for engineers, ensuring that only qualified engineers are onboarded.

## Usage

1. Create an `Engineer` object with the required attributes (name, portfolio, experience, skills).
2. Call the `vetting_process` function with the `Engineer` object as an argument.
3. The function will return a string indicating whether the engineer was onboarded successfully or not.
