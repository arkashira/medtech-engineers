# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute**: $15/user/month
- **Storage**: $5/user/month
- **Bandwidth**: $3/user/month  
**Total Cost per Active User**: **$23/user/month**

### Pricing Tiers
| Tier            | Price ($/mo) | Features                                                                                     |
|-----------------|---------------|----------------------------------------------------------------------------------------------|
| **Basic**       | $99           | Access to pre-trained engineers, basic support, and project management tools.               |
| **Professional**| $199          | All Basic features + dedicated project manager, advanced analytics, and priority support.    |
| **Enterprise**   | $399          | All Professional features + custom solutions, SLA guarantees, and 24/7 support.             |

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: **$200 - $400** per user

### Lifetime Value (LTV) Estimate
- Average user retention: 24 months
- Average Revenue per User (ARPU): $199 (assuming Professional tier is the most popular)  
**LTV = ARPU × Retention = $199 × 24 = $4,776**

### Break-even Users Count
- Fixed Costs (monthly): $5,000 (marketing, salaries, operational costs)
- Contribution Margin per User = Price - Cost = $199 - $23 = $176  
**Break-even Users = Fixed Costs / Contribution Margin = $5,000 / $176 ≈ 29 users**

### Path to $10K MRR
- Target MRR: $10,000
- Average Revenue per User (ARPU) from Professional Tier: $199  
**Users Required = Target MRR / ARPU = $10,000 / $199 ≈ 50 users**  
- If targeting Basic Tier ($99):  
**Users Required = $10,000 / $99 ≈ 101 users**  
- If targeting Enterprise Tier ($399):  
**Users Required = $10,000 / $399 ≈ 26 users**  

### Summary
- **Break-even Users Count**: 29 users
- **Path to $10K MRR**: 
  - **Professional Tier**: 50 users
  - **Basic Tier**: 101 users
  - **Enterprise Tier**: 26 users
